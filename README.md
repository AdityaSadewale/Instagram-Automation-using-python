# 📸 Instagram Automation Using Python 🤖

This project demonstrates how to automate basic Instagram actions using **Python** and the **Instabot** library.  
With this script, you can log in, follow/unfollow users, send messages, and fetch followers data automatically.

---

## 🚀 Features

✅ Login to Instagram  
✅ Follow & unfollow users  
✅ Send direct messages 📩  
✅ Upload photos 🖼️  
✅ Get followers & following list  
✅ Fetch detailed user information  

---

## 🛠️ Technologies Used

- 🐍 Python 
- 🤖 Instabot
- 🌐 Instagram API (via Instabot)
